Installation Guide

Prerequisites

  • Android devices or emulators (Android 7+)
  • ADB/Appium setup for device communication
  • Node.js 16+ and Python 3.8+ (if applicable)
  • USB debugging or Wi-Fi ADB connection
  • Accessibility service permissions

Quick Setup

1. Clone Repository

git clone https://github.com/Appilot123/Android-ig-Follow-Unfollow-Bot.git

2. Configure Environment

cp .env.example .env && nano .env

3. Install Dependencies

npm install && pip install -r requirements.txt

4. Connect Devices

adb devices

5. Start Scheduler

npm run start:scheduler

6. Monitor Dashboard

npm run dashboard

For complete setup instructions, configuration options, and troubleshooting, see the detailed README documentation.

Frequently Asked Questions

Is this Instagram follow unfollow bot safe to use?

Yes, when used with conservative settings. Our bot achieves <1% block rates through human-like pacing, randomized delays, and intelligent cooldowns. Always follow Instagram's Terms of Service and start with low daily limits.

What are the recommended daily activity caps?

Start with 50-100 follows per day for new accounts, gradually increasing to 200-300 for aged accounts. Unfollow limits should match follow volumes. Always monitor account health and adjust accordingly.

Does this auto unfollow bot instagram work on emulators?

Yes, fully compatible with Android emulators including BlueStacks, NoxPlayer, and Genymotion. Emulators offer easier scaling but may have slightly different performance characteristics than real devices.

How many devices can I run simultaneously?

The bot scales from single devices to 50+ device deployments. Performance depends on your hardware resources and network capacity. Start small and scale based on your infrastructure.

How do retries and backoff work?

The bot implements exponential backoff for failed actions, temporary cooldowns for rate limit responses, and intelligent retry logic that adapts to Instagram's feedback patterns to maintain account safety.

Can I whitelist certain accounts from unfollowing?

Yes, comprehensive whitelist support allows you to protect valuable connections, verified accounts, or custom user lists from automated unfollowing actions.

How do I avoid getting blocked or detected?

Use conservative daily limits, enable all human-like features, rotate activity patterns, take regular breaks, and monitor account health. The bot includes built-in protections, but responsible usage is essential.

What logging and reporting features are available?

Comprehensive logging includes action logs, error screenshots, JSON reports, performance metrics, and a web dashboard with real-time monitoring and historical analytics.
